APeX:

Epi Pens

Non-Formulary in BCH, used in adults. Can change dispensable product at verification if epipen is accidentally ordered.

Heparin Infusion Indication

Change to indication being set up more like antibiotics. Right now old pedi language will be defaulted in as “other” indication in ordersets. Trying to see if we can come to an agreement with adults on real indications.

K-Centra Order Set

P&T approved, not FDA approved for peds at this time but may need to use adult orderset once finalized.

Medication Nursing Policy & Procedures (IVIG, Remicade etc.)

Nursing requesting policies/procedures for meds they don’t give frequently but have special infusion rates, monitoring or other considerations. Minh working with RN supervisors to create standardized format for these and will reach out to various other RPh’s as needed for assistance. Will need final approval from BCH Medication committee as well. Will get them posted online as well once completed.

Ammonul

Working on fix to calculate mixture. Current possible fix will include pharmacist needing to delete some of the drug name to remove mL of D10W

  • To promote med rec, admission ordersets are only available in admission navigator for providers. But now pharmacists cannot preview admission ordersets.
  • To change login context (can allow you to view how a provider sees various things but not all things show up like provider- sometimes pharmacist settings will override), click on Epic button and pick ‘change context’. Inpatient login contexts have “SVC” in their name (ie- PED HOSPITAL MED SVC)

Sent one lamictal and one labetalol tablet (intended to be two lamictal tablets). Not on our confused drug name list but is on Lexi one. Elham seeing if we should add to our list.

PCA ordered not on order set/no auxiliary meds/monitoring parameters

Will do further investigation and ask Willow how they we able to order this as well. Reminder to staff that all PCAs must be ordered via ordersets.

Custom IVF with K Phos, admin instructions - (No Filter Used)
